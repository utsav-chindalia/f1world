# F1 Racing Educator Prototype

An interactive F1 racing education platform built with Phaser.js and Vite.

## Description

F1 Racing Educator is a web-based educational platform designed to help users learn about Formula 1 racing through interactive experiences and simulations.

## Prerequisites

- Node.js (v14 or higher recommended)
- npm (comes with Node.js)

## Installation

1. Clone the repository:
```bash
git clone [your-repository-url]
cd f1-edu-racer-prototype
```

2. Install dependencies:
```bash
npm install
```

## Development

To start the development server:

```bash
npm run dev
```

This will start the development server at `http://localhost:5173` (or another port if 5173 is in use).

## Building for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` directory.

To preview the production build:

```bash
npm run preview
```

## Tech Stack

- [Vite](https://vitejs.dev/) - Next Generation Frontend Tooling
- [Phaser](https://phaser.io/) - HTML5 Game Framework
